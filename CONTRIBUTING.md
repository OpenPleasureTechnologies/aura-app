# Contributing to AURA 𖦹

Thank you for being here. You are welcome.

## Anyone can contribute

**No coding?** Test devices, write docs, design, translate, or just share AURA.

**Beginner coder?** Look for issues tagged `good-first-issue`

**Expert?** Review PRs, optimize BLE, improve ML models

## How to start

1. Fork the repo
2. Create a branch: `git checkout -b feature/your-feature`
3. Commit: `git commit -m "Add your feature"`
4. Push: `git push origin feature/your-feature`
5. Open a Pull Request

## Code style

- Kotlin: Follow official Kotlin conventions
- Use meaningful variable names
- Comment complex logic

## Pull Request checklist

- [ ] Code compiles
- [ ] Tested on real device (if hardware-related)
- [ ] No sensitive data logged
- [ ] Privacy preserved

## Questions?

Open an issue or comment on existing ones.

We're building together. 🧡
